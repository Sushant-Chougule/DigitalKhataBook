# Feature Request Template - DigitalKhataBook

Use this template when requesting new features from Gemini CLI.

---
## 1. Feature Name
*Example: Email Notifications*

## 2. Business Purpose
*Explain why this feature is needed and what it solves.*

## 3. Affected Pages
*List the pages (e.g., Dashboard, Customer Profile) that will change.*

*Are there new URL endpoints? (Example: POST /customers/:id/send-email).*

## 6. Logic Requirements
*Describe specific calculations or business rules.*
